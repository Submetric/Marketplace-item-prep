# Getting Started

Marketplace Item Prep helps you prepare Marketplace avatar item thumbnails faster inside Roblox Studio.

This guide covers the current public `Lite` and `Pro` workflows.

## Before You Start

Marketplace Item Prep is meant for creator-side thumbnail prep.

It currently works best when you start from a selected:

- `Accessory`
- `Model`
- `MeshPart`

The plugin is designed for scene setup, framing, and review. It is not a replacement for Roblox fitting, full validation, or Marketplace publishing tools.

## First 60 Seconds

1. Install the plugin.
2. Select the item you want to prepare.
3. Open Marketplace Item Prep.
4. Start in `World Shot`, or switch to `Stage Shot` if you want a cleaner preview scene.
5. Pick a camera preset.
6. Press `Frame` to get a stable composition quickly.
7. Adjust background and lighting.
8. If needed, use `Reset` or clean the stage.

## World Shot

`World Shot` works in your current place context.

Use it when:

- the current scene is already usable
- you want to keep the world background visible
- you want a faster baseline thumbnail pass

## Stage Shot

`Stage Shot` creates an isolated plugin-owned preview scene for focused composition work.

Use it when:

- the current place is visually noisy
- you want a cleaner backdrop
- you want more controlled presentation work

During preview, the plugin works through its own preview session instead of silently changing your source asset.

## Core Lite Controls

The `Lite` workflow includes:

- valid target detection
- camera presets
- `Frame`
- `Reset`
- basic backdrop swatches
- basic lighting swatches
- stage cleanup and return actions
- last-used settings

Recommended order:

1. choose a preset
2. press `Frame`
3. adjust the scene
4. only then move into deeper refinement if you need it

## Pro Extras

`Pro` includes everything in `Lite`, then adds:

- `Quick Compose` for faster composition steering
- `Fine Tune` camera controls for precise adjustment
- full background palette access
- advanced lighting controls
- review candidate apply
- `Scene Presets`
- `Safe Framing Guardrail`

`Pro` is built for stronger single-item composition, cleaner art direction, and safer framing decisions on the current item.

## Common First-Time Mistakes

- starting with an invalid selection
- skipping `Frame` and over-adjusting too early
- expecting `Stage Shot` to behave like a publish or export flow
- assuming `Pro` is a batch workflow tier instead of a single-item quality tier
