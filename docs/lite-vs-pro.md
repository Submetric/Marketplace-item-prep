# Lite vs Pro

Marketplace Item Prep is currently documented in two public tiers:

- `Lite`: free baseline for fast thumbnail prep
- `Pro`: paid single-item quality tier

## Overview

`Lite` is meant to be useful immediately.

`Pro` builds on that baseline with deeper control, stronger composition tools, and lower shot-risk on the current item.

## Feature Comparison

| Feature | Lite | Pro |
| --- | --- | --- |
| Valid target detection | Included | Included |
| `World Shot` | Included | Included |
| `Stage Shot` | Included | Included |
| Camera presets | Included | Included |
| `Frame` / `Reset` | Included | Included |
| Basic backdrop swatches | Included | Included |
| Basic lighting swatches | Included | Included |
| Last-used settings | Included | Included |
| `Quick Compose` | No | Included |
| `Fine Tune` camera controls | No | Included |
| Full background palette | No | Included |
| Advanced lighting controls | No | Included |
| Review candidate apply | No | Included |
| `Scene Presets` | No | Included |
| `Safe Framing Guardrail` | No | Included |

## Lite

Best for:

- first-time evaluators
- occasional creators
- creators who want a simple free baseline

What it does well:

- fast thumbnail prep
- clean `World Shot` / `Stage Shot` switching
- quick framing and scene adjustment
- non-destructive preview workflow

## Pro

Best for:

- creators who want more control over the current item
- creators who care about cleaner art direction
- creators who want safer framing and more reusable single-item setups

What it adds:

- stronger camera control
- broader background and lighting control
- reviewed shot selection
- named scene reuse
- advisory framing guidance

## Which Version Should I Choose

Choose `Lite` if you want:

- a free way to prepare thumbnails faster
- a cleaner baseline workflow
- a simple Studio-native starting point

Choose `Pro` if you want:

- better control over composition and camera cleanup
- more polish in the final shot
- scene reuse through `Scene Presets`
- safer crop and framing guidance through `Safe Framing Guardrail`

## Important Scope Note

`Pro` is focused on improving the current item.

It is not positioned as a multi-item workflow, submit-ready, or publish-adjacent tier in the current public docs.
